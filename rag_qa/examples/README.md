### 2️⃣ 交互式问答

**文件**: `examples_02_interactive_qa.py`

**功能**: 启动 RAG 问答系统（支持多轮对话）

**使用方法**:
```bash
python examples_02_interactive_qa.py
```

**交互命令**:
- 直接输入问题 → 开始问答
- `clear` → 清空对话历史
- `history` → 查看最近对话
- `exit` / `quit` / `q` → 退出系统

**输出示例**:
```

================================================================================

正在初始化系统...
✅ Milvus 连接成功

  - 输入问题开始问答
  - 输入 'clear' 清空对话历史
  - 输入 'history' 查看当前对话历史
  - 输入 'exit' 或 'quit' 退出系统
================================================================================

--------------------------------------------------------------------------------

1. 穿孔爆破：使用牙轮钻机、潜孔钻机等设备进行穿孔作业...
2. 采装作业：用挖掘机将爆破后的矿石装入运输车辆...
3. 运输作业：通过矿用卡车或皮带输送机运送矿石...
4. 排土作业：将剥离的废石运至排土场...
```

### 3️⃣ 知识库溯源

**文件**: `examples_03_view_sources.py`

**功能**: 查看检索结果的来源信息

**使用方法**:
```bash
# 交互模式（推荐）
python examples_03_view_sources.py -i

# 基础演示
python examples_03_view_sources.py --demo

# 导出演示
python examples_03_view_sources.py --export-demo
```

**交互命令**:
- 直接输入问题 → 查看溯源
- `export json` → 导出为 JSON
- `export csv` → 导出为 CSV
- `export md` → 导出为 Markdown
- `stats` → 显示统计信息
- `exit` → 退出

## 💡 自定义示例

基于这些示例，你可以轻松创建自己的脚本：

```python
# my_custom_script.py
from core.vector_store import VectorStore
from core.new_rag_system import RAGSystem

# 初始化
vector_store = VectorStore()
rag_system = RAGSystem(vector_store, call_llm)

# 执行特定任务
answers = list(rag_system.generate_answer(query="我的问题"))
print(answers[0])
```
